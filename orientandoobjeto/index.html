Algumas definições


A orientação a objetos no JavaScript é uma paradigma de programação que permite a criação de classes, objetos e a definição de suas propriedades e métodos. Embora o JavaScript seja uma linguagem de programação baseada em protótipos, ele também oferece suporte á orietação a objetos, possibilitando a criação de estruturas mais complexas e reutilizáveis.

As variáveis JavaScript podem armazenar vários valores usando uma estrutura de dados chamada "arrays". Um array é uma coleção ordenada de valores que podem ser diferentes tipos, como números, stings, objetos, etc.

As propriedades do objeto no JavaScript são pares chave-valor que representam carcateristícas ou atributos do objetoo. Cada objeto pode conter várias propriedades e cada propriedade é identificada por uma chave única. As propriedades podem ser de diferentes tipos, como string, números, booleanos, funções, outros objetos, etc.
